# raspberry-pi-pico-based-handheld-device-project
A Raspberry Pi Pico–powered handheld gaming and utility device featuring multiple interactive games, a colorful UI on the ILI9341 TFT display, and buzzer feedback. Created for Waves 2025 “Create in India Challenge,” it was selected among the Top 7 teams nationally for innovation, creativity, and excellence in embedded systems design.
